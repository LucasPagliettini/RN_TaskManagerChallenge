# TODO React Native App Challenge

This is an APP that lets the user register and handle tasks. 

It has a tab navigation menu that lets the user see the whole tasks list, and also see only completed tasks, uncompleted tasks or favorite tasks. 
Each listed task has a checkBox that tells the user if the task is completed o not and also lets the user to change that state. When it changes, the tasks lists are automatically updated.
